
# -_15## Практическая работа №15: Хранение изображений (BLOB)

### Описание изменений
В структуру базы данных `EventOrganizerDB` внесены следующие изменения:
1. В ключевую таблицу `Events` добавлено поле `Image` типа `VARBINARY(MAX)` для хранения бинарных данных (изображений мероприятий).
2. Скрипт инициализации базы данных (`script.sql`) обновлен для создания новой колонки.
3. Все существующие записи в таблице `Events` заполнены тестовыми бинарными данными (заглушками JPEG).

### Технические детали
- Тип данных: `VARBINARY(MAX)` позволяет хранить файлы размером до 2 ГБ.
- Для целей тестирования использованы минимальные заголовки файлов JPEG.
